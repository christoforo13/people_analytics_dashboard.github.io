# People Analytics Dashboard

Este proyecto presenta un dashboard desarrollado con el objetivo de formar parte de un portafolio profesional orientado al área de visualización de datos.  
El análisis está enfocado en métricas de People Analytics, utilizando visualizaciones para explorar temas como rotación, satisfacción laboral, desempeño y características generales de los empleados.

## Objetivo del Proyecto

El propósito de este dashboard es demostrar habilidades en:

- Construcción y limpieza de datos.
- Desarrollo de KPIs relevantes para análisis de personal.
- Diseño de visualizaciones en Power BI.
- Presentación clara de insights en un entorno profesional.

## Contenido del Dashboard

El dashboard está organizado en varias páginas, cada una con un enfoque distinto dentro del análisis de personal.

### 1. Overview
Página que resume indicadores generales relacionados con:

- Rotación de personal.
- Horas extra.
- Distribución de empleados.
- KPIs principales.

### 2. Attrition Analysis
Página dedicada al análisis de rotación, incluyendo:

- Comparaciones por departamentos.
- Relaciones con overtime.
- Distribución por edad y salario.
- Factores que influyen en la rotación.

### 3. Employee Profile
Página enfocada en describir la composición del personal:

- Educación.
- Satisfacción laboral.
- Work-life balance.
- Calificaciones de desempeño.
- Otros indicadores cualitativos.

Las imágenes del dashboard están disponibles en la carpeta `/images`.

## Tecnologías Utilizadas

- Power BI para modelado, KPIs y visualizaciones.
- DAX para creación de medidas.
- GitHub Pages para publicación del portafolio.
- HTML para la presentación en línea del proyecto.
